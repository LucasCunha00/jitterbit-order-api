# Jitterbit Order API

API simples desenvolvida em **Node.js com Express** para gerenciamento de pedidos.

## Funcionalidades

A API permite:

- Criar pedidos
- Buscar pedido pelo número
- Listar todos os pedidos
- Deletar pedidos

## Tecnologias utilizadas

- Node.js
- Express
- JSON
